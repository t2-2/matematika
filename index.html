פרק ראשון

מה זו
משוואה ריבועית?

משוואה ריבועית היא משוואה שבה הנעלם x מופיע בחזקה שנייה — כלומר יש לנו x².

הצורה הכללית

כל משוואה ריבועית ניתנת לכתיבה כך:

ax² + bx + c = 0

כש‑a, b, c הם מספרים כלשהם, ו‑a ≠ 0 (כי אם a=0 היא כבר לא ריבועית)

  • ✓

    x² − 5x + 6 = 0

    כאן: a=1, b=−5, c=6. משוואה ריבועית קלאסית.

  • ✓

    2x² + 3x = 0

    גם זו ריבועית — a=2, b=3, c=0.

  • ✓

    ½x² − ⅔x + ⅙ = 0

    גם זו ריבועית! רק שה‑a, b, c הם שברים — ובדיוק זה נלמד לפתור.

💡 הפתרונות של משוואה ריבועית נקראים שורשים. לרוב יש שניים, לפעמים אחד, ולפעמים אין בכלל (ממשיים).
2
פרק שני

למה שברים
מסבכים את החיים?

כשיש שברים במשוואה, חישובים נהיים מסורבלים. למשל:

12x² − 23x + 16 = 0

קשה לעבוד עם זה ישירות. הטריק המרכזי הוא להיפטר מהשברים לגמרי — עוד לפני שמתחילים לפתור.

מה זה מכנה משותף (מ"מ)?

המכנה המשותף הקטן ביותר (ל"מ) הוא המספר הקטן ביותר שמתחלק בכל המכנים של השברים.


למשוואה שלנו המכנים הם: 2, 3, 6

ה‑ל"מ שלהם הוא: 6

(כי 6 = 2×3, ו‑6 מתחלק ב‑2, ב‑3, וב‑6)

🎯 האסטרטגיה: נכפיל את כל המשוואה בל"מ — וכל השברים ייעלמו!

פרק רביעי · פתרון אינטראקטיבי

פתרון צעד אחר צעד

בואו נפתור יחד:

½x² − ⅔x + ⅙ = 0
⬡   SOLVER INTERACTIVE שלב 1 / 5
שלב 1 — מציאת ל"מ

מזהים את כל המכנים: 2, 3, 6.
המספר הקטן שמתחלק בכולם הוא 6.

ל"מ = 6
שלב 2 — כפל כל המשוואה ב‑6

נכפיל כל איבר בנפרד:

6 · ½x²  −  6 · ⅔x  +  6 · ⅙ = 6 · 0

↓   מפשטים

3x²  −  4x  +  1 = 0

✓ אין יותר שברים! עכשיו יש לנו משוואה ריבועית "רגילה".

שלב 4 — הפרדה לגורמים

נסה להפריד: 3x² − 4x + 1 = (3x − 1)(x − 1)

אימות: (3x−1)(x−1) = 3x²−3x−x+1 = 3x²−4x+1 ✓

(3x − 1)(x − 1) = 0

מכיוון שמכפלה = 0, אחד הגורמים חייב להיות אפס:

3x − 1 = 0
x = ⅓
x − 1 = 0
x = 1
✓ פתרון

הפתרונות של המשוואה ½x² − ⅔x + ⅙ = 0 הם:

x₁ = ⅓     x₂ = 1

אפשר לאמת: מציבים x=1: ½(1)−⅔(1)+⅙ = 3/6−4/6+1/6 = 0 ✓
מציבים x=⅓: ½(⅑)−⅔(⅓)+⅙ = 1/18−2/9+1/6 = 1/18−4/18+3/18 = 0 ✓

🎉 הצלחת!

פרק חמישי

הנוסחה
הריבועית

לא תמיד אפשר להפריד לגורמים בקלות. לכן קיימת הנוסחה הריבועית — שפותרת כל משוואה ריבועית.

QUADRATIC FORMULA
x = −b ± √(b²−4ac) 2a

x = (−b ± √(b² − 4ac)) / 2a

  • a

    המקדם של x²

    הכי חשוב — חייב להיות שונה מאפס.

  • b

    המקדם של x

    שים לב לסימן! אם המשוואה כוללת −5x אז b = −5.

  • c

    האיבר החופשי

    המספר ללא x. יכול להיות גם 0.

  • Δ

    הדיסקרימיננטה: b² − 4ac

    Δ > 0 → שני פתרונות שונים
    Δ = 0 → פתרון יחיד (כפול)
    Δ < 0 → אין פתרון ממשי

פרק שישי · דוגמה

שימוש בנוסחה
בפועל

נפתור את המשוואה:

¼x² + ½x − 2 = 0
שלב 1 — ל"מ = 4, כפל המשוואה ב‑4
4 · ¼x² + 4 · ½x − 4 · 2 = 0
x² + 2x − 8 = 0
שלב 2 — זיהוי a, b, c

a = 1,   b = 2,   c = −8

שלב 3 — חישוב דיסקרימיננטה

Δ = b² − 4ac = (2)² − 4·1·(−8) = 4 + 32 = 36

√36 = 6  →  יש שני פתרונות ✓

שלב 4 — הנוסחה
x = (−2 ± 6) / 2
x₁ = (−2+6)/2 = 4/2 = 2
x₂ = (−2−6)/2 = −8/2 = −4
